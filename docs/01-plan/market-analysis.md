## 2. 국내 경쟁 제품 분석

### 2.1 알씨 (ALSee) — ESTsoft

| 항목 | 내용 |
|------|------|
| 개발사 | 이스트소프트 |
| 출시 | 2000년 (25년 역사) |
| 사용자 | 2,500만+ (알툴즈 패키지 기준) |
| 가격 | 개인 무료 / 기업 유료 |
| 최신 버전 | v9.51 (2025) |

**지원 기능**
- 이미지 보기: 크게보기, 연속보기, 전체화면, 여럿보기
- 편집: 자르기, 보정, 필터, 효과, 스티커, 액자
- **AI 기능 (2025 신규)**: AI 배경 제거, AI 화질 개선, AI 얼굴 모자이크
- CAD 뷰어: DWG, DWF, DXF 지원 (특이점)
- 포맷 변환: BMP, GIF, JPG, TIF, PCX, PNG
- 압축파일 내 이미지 보기: ZIP, RAR, 7Z
- 클라우드 연동 (알씨 클라우드)
- GIF/동영상 만들기

**장점**
- ✅ 국내 1위 브랜드 인지도 (알툴즈 패키지)
- ✅ 25년 누적 사용자 기반
- ✅ AI 기능 적극 탑재로 차별화
- ✅ CAD 뷰어 내장 (유일한 특화 기능)
- ✅ 한국어 UI 최적화

**단점**
- ❌ 알툴즈 번들 설치 유도 (광고성 번들)
- ❌ 무거운 리소스 (많은 기능 = 높은 메모리)
- ❌ 기업 사용 유료 (라이선스 구매 필요)
- ❌ Windows 전용
- ❌ AI 기능 사용 횟수 제한 (로그인 필요)

### 2.2 꿀뷰 (HoneyView) — 반디소프트 ⚠️단종

| 항목 | 내용 |
|------|------|
| 개발사 | 반디소프트 |
| 출시 | 2014년 |
| 단종 | 2023년 9월 (개발 중단) |
| 후속 | 반디뷰 (BandiView) |

**특징**
- 심플하고 가벼운 것이 핵심 가치
- 압축 파일(ZIP, RAR, CBZ 등) 내 이미지 바로 보기
- 만화/웹툰 독자들에게 인기 높았음
- BMP, JPG, GIF, PNG, PSD, DDS, WebP 지원

> 꿀뷰의 단종은 **가벼운 무료 뷰어 시장에 공백**이 생겼음을 의미.
> 반디뷰가 이를 유료로 대체하면서 무료 경량 뷰어 자리가 비어있음.

### 2.3 반디뷰 (BandiView) — 반디소프트

| 항목 | 내용 |
|------|------|
| 개발사 | 반디소프트 |
| 포지션 | 꿀뷰 후속 (유료 전환) |
| 가격 | 개인 비상업용 무료 / 상업용 유료 |
| 최신 버전 | v7.21 |

**지원 기능**
- HDR 출력 지원
- RAW 파일 지원 (최신 카메라)
- HEIC 파일 지원 (iPhone)
- Clip Studio Paint 파일 지원
- **웹툰 보기 모드** (세로 연결 이미지)
- 이미지 변환 (10배 빠른 배치 처리)
- 압축 파일 내 이미지 보기 (반디집 연동)
- HiDPI 지원

**장점**
- ✅ 꿀뷰 대비 대폭 향상된 포맷 지원 (RAW, HEIC)
- ✅ 웹툰 모드 (국내 틈새 시장)
- ✅ HDR 출력 (고급 디스플레이 지원)
- ✅ 빠른 성능

**단점**
- ❌ 꿀뷰 시절의 완전 무료에서 유료 전환으로 이탈 사용자 발생
- ❌ 상업적 사용 유료
- ❌ macOS 미지원
- ❌ AI 기능 없음

### 2.4 "빵씨" 관련

> **조사 결과**: "빵씨"라는 이름의 공식 이미지 뷰어 제품은 검색 결과에서 확인되지 않음.
> 가능성: 커뮤니티 내 별칭, 소규모 배포 소프트웨어, 또는 혼동된 이름일 수 있음.

## 5. 시장 기회 분석

### 5.1 명확한 시장 공백

```
[공백 1] 꿀뷰 단종으로 생긴 "완전 무료 + 경량 + 깔끔한 UI" 자리
         → 꿀뷰 사용자들이 대안을 찾고 있음

[공백 2] 크로스플랫폼 (Win + Mac) 지원하는 경량 뷰어 부재
         → IrfanView/FastStone/알씨/반디뷰 모두 Windows 전용

[공백 3] AI 기능 없는 진짜 가벼운 뷰어 (알씨의 AI = 무거워짐)
         → 빠르고 가벼운 뷰어만 원하는 사용자층 이탈

[공백 4] Rust 기반 네이티브 성능 뷰어 없음
         → egui + image crate = IrfanView 수준의 속도 + 현대적 UI
```

### 5.2 타겟 사용자

| 세그먼트 | 설명 | 규모 |
|---------|------|------|
| 꿀뷰 이탈자 | 반디뷰 유료 전환에 불만족한 사용자 | 대 |
| 개발자/파워유저 | 빠르고 가벼운 도구 선호 | 중 |
| 만화/웹툰 독자 | 압축 파일 이미지 보기 필요 | 중 |
| macOS 사용자 | 한국어 지원 경량 뷰어 없음 | 소 |
| 기업 IT 담당자 | 무료 상업용 라이선스 원함 | 중 |

### 5.3 RustView 차별화 포인트

```
1. 완전 무료 (MIT 라이선스) — 상업용 포함
   → 알씨/반디뷰/FastStone은 상업용 유료

2. Windows + macOS 동시 지원
   → 국내 경쟁사 모두 Windows 전용

3. 단일 바이너리, ~10MB 이하
   → 설치 불필요, USB 포터블 가능

4. 빠른 시작 (<500ms)
   → MS Photos 대비 체감 속도 압도적

5. 깔끔한 현대적 UI (egui)
   → IrfanView의 구식 UI 대비 우위

6. 오픈소스 투명성
   → 광고/번들 없음 보장
```
