## 1. 디자인 시스템

### 컬러 팔레트 (Dark Mode First)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--bg-base` | `#0f0f10` | 뷰어 배경 |
| `--bg-surface` | `#1a1a1d` | 툴바, 패널 배경 |
| `--bg-elevated` | `#242428` | 버튼 hover, 썸네일 배경 |

| `--text-secondary` | `#9898a8` | 보조 텍스트 |
| `--text-muted` | `#55555f` | 비활성/힌트 텍스트 |

## 2. 화면 구성

### 2.1 앱 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  TOOLBAR (44px)                                         │
│  🦀 [📂] [‹] [›] | filename.jpg    [−][100%][+][맞춤][1:1] │
│                          [↺][↻] | [뷰어|갤러리] | [ℹ][⛶] │
├─────────────────────────────────────────────┬───────────┤
│                                             │ INFO      │
│             VIEWER / GALLERY                │ PANEL     │
│                                             │ (260px)   │
│                                             │           │
├─────────────────────────────────────────────┴───────────┤
│  STATUSBAR (26px)  🖼 4/24  📐 6016×4016  🔍 100%        │
└─────────────────────────────────────────────────────────┘
```

### 2.2 뷰어 화면

| 요소 | 설명 |
|------|------|
| 이미지 영역 | 중앙 정렬, 마우스 휠 줌, 드래그 팬 |
| 좌/우 화살표 | hover 시 표시, 반투명 유리 효과 |
| 줌 뱃지 | 뷰어 hover 시 하단 중앙 표시 |
| 드래그앤드롭 | 파일 드롭 시 점선 테두리 표시 |

### 2.3 갤러리 화면

| 요소 | 설명 |
|------|------|
| 폴더 바 | 현재 경로, 정렬 옵션, 총 개수 |
| 썸네일 그리드 | auto-fill (최소 140px), 1:1 비율 |
| 선택 효과 | accent 색상 테두리 |
| hover 효과 | 파일명 라벨 표시, 약간 scale-up |

### 2.4 정보 패널 (260px)

| 섹션 | 내용 |
|------|------|
| 미리보기 썸네일 | 16:9 비율 |
| 파일 정보 | 파일명, 포맷, 크기, 해상도, 수정일 |
| EXIF 정보 | 카메라, 렌즈, 셔터, 조리개, ISO, 초점거리 |
| 폴더 정보 | 경로, 폴더 내 순번 |

## 3. 인터랙션 명세

| 트리거 | 동작 |
|--------|------|
| `←` `→` 키 | 이전/다음 이미지 |
| `+` `-` 키 | 줌 인/아웃 (×1.25 단계) |
| `0` 키 | 창 맞춤 |
| `I` 키 | 정보 패널 토글 |
| `G` 키 | 갤러리/뷰어 전환 |
| `F` 키 | 전체화면 |
| `L` / `R` 키 | 좌/우 회전 |
| 마우스 휠 | 줌 인/아웃 |
| 드래그 팬 | 이미지 이동 (확대 시) |
| 파일 드롭 | 이미지 열기 |
| 갤러리 클릭 | 뷰어로 전환 + 해당 이미지 표시 |

## 4. egui 컴포넌트 매핑

| 목업 요소 | egui 구현 |
|----------|-----------|
| Toolbar | `TopBottomPanel::top` + `egui::menu::bar` |
| Viewer | `CentralPanel` + `egui::Image` |
| Nav arrows | `egui::Button` (절대 위치) |
| Info panel | `SidePanel::right` |
| Gallery grid | `ScrollArea` + `Grid` |
| Status bar | `TopBottomPanel::bottom` |
| Zoom badge | `egui::Area` (floating) |

## 6. Phase 6 구현 시 주의사항

- egui의 `Image` 위젯은 `TextureHandle`로 GPU 텍스처 업로드 필요
- 갤러리 썸네일은 별도 스레드에서 비동기 로딩 (`rayon` 사용)
- 정보 패널의 EXIF는 `kamadak-exif` 크레이트 활용
- 키보드 단축키는 `ctx.input(|i| i.key_pressed(...))` 패턴
